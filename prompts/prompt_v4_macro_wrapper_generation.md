# Prompt V4 – Macro Wrapper Generation

## Objective

Generate a Verilog wrapper for the AMUX2_3V analog macro using AI assistance.

## Prompt

Generate a synthesizable Verilog wrapper for an analog 2:1 multiplexer macro named AMUX2_3V.

Requirements:

* Preserve analog macro interface
* Suitable for OpenLane mixed-signal integration
* Black-box compatible
* Include input/output declarations
* Avoid internal implementation details

## Expected Outcome

A wrapper that can be referenced during synthesis while the actual analog implementation remains external.
