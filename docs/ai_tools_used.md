# AI Tools Used

## ChatGPT (GPT-5)

### Purpose

Used as an engineering assistant throughout the project.

### Applications

* Repository understanding
* Mixed-signal flow explanation
* Analog macro study
* LEF/GDS interpretation
* OpenLane configuration understanding
* Debugging assistance
* Error root-cause analysis
* Documentation generation
* Report interpretation

### Verification

All AI-generated suggestions were manually validated before implementation.

---

## AI-Assisted Workflow

The project followed the workflow:

Problem Understanding
↓
Prompt Creation
↓
AI Analysis
↓
Manual Verification
↓
Implementation
↓
Debugging
↓
Result Validation

---

## Key Observation

AI significantly reduced the time required to understand unfamiliar tools and debugging workflows.

However, engineering judgement and manual verification remained essential throughout the project.

The final implementation results were accepted only after successful OpenLane execution and verification.
